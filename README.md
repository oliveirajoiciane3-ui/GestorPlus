## Objetivo do Software
O sistema GestorPlus tem como objetivo auxiliar pequenas e médias empresas no gerenciamento de clientes, produtos e pedidos, oferecendo uma interface simples, organizada e eficiente para controle das informações.

## Funcionalidades do Sistema
- Cadastro de clientes
- Cadastro de produtos
- Registro de pedidos
- Consulta de clientes cadastrados
- Consulta de produtos cadastrados
- Integração com banco de dados MySQL

## Banco de Dados
O sistema utiliza banco de dados MySQL para armazenar as informações de clientes, produtos e pedidos.

## Observações
Projeto desenvolvido como parte do Projeto Integrador utilizando Java no NetBeans com conexão ao MySQL.
